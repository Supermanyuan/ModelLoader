For this project we will be designing and creating software to view 3D CAD models and simulate. It will be created using the QT and VTK libraries and written using C++. The program will be available for both Windows and Mac OS

**ROLES**

- Vector class: Hana
- Model class: Ed
- Cell class: Harry
- Material class: P

This repository will include 4 classes (.cpp and .h files) as well as a main.cpp file combining all classes and integrating them. The repository will also include several test files for each class as well as one for the main.cpp file. .a file will be included to compile as well as a cmake file A .gitignore file is included
